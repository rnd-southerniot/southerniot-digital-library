# Modbus RTU Node (RAK4630 + RAK5802) for Selec MFM384

## Purpose
Reads Selec MFM384 values over RS-485 Modbus RTU and uplinks telemetry via LoRaWAN.

## Defaults (Locked)
- Region: `AS923-1`
- Modbus RTU: `9600-N-8-1`

## Field Notes
- Evidence photos are required at `site_survey`, `on_site_installation`, and `testing` (4–5 photos with tags).
- RS-485 wiring must be verified (A/B polarity, termination where required).
- Attach test evidence (screenshots/photos/logs) back to the CRM workstream.
