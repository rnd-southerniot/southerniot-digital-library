# Selec MFM384 - Meter Setup (Field + Lab)

## Default Modbus Settings (SouthernIoT Standard)
- Baudrate: **9600**
- Parity: **None (N)**
- Data bits: **8**
- Stop bits: **1**
- Slave ID: **1** (changeable per site)

## Required checks
- Confirm RS-485 A/B labeling on the meter
- Confirm Modbus address/slave ID
- Confirm CT/PT ratios configured correctly (affects scaling)
