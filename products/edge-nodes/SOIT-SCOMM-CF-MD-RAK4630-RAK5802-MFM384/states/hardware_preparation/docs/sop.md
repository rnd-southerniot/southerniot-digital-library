# SOP: hardware_preparation (RAK4630 + RAK5802 + MFM384)

## Goal
Assemble and label node hardware for the site.

## Steps
1. Inventory: RAK4630 core, RAK5802 RS-485 baseboard, enclosure, glands, cables.
2. Label the enclosure with `project_id` and meter identifier.
3. Pre-configure Modbus settings (must be `9600-N-8-1`).
4. Confirm meter `slave_id` plan (default `1` unless site requires otherwise).

## Outputs
- Prepared hardware list attached to CRM
