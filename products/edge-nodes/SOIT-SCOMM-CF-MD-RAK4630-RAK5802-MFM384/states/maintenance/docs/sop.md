# SOP: maintenance (RAK4630 + RAK5802 + MFM384)

## Goal
Review node health, data quality, and planned changes.

## Steps
1. Confirm last-seen, battery/power status (if applicable), and link quality.
2. Review alerts and field issues; open corrective tasks.
3. If Modbus mapping changes are needed, update the library and rerun validation.

## Outputs
- Maintenance log entry in CRM
