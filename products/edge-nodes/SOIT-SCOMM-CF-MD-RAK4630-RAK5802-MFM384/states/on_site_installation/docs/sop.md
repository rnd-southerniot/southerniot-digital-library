# SOP: on_site_installation (RAK4630 + RAK5802 + MFM384)

## Goal
Install the node and connect RS-485 to the MFM384 safely and correctly.

## Steps
1. Confirm meter terminals and isolation/safety requirements.
2. Mount enclosure and route RS-485 cable away from high-noise runs.
3. Connect A/B and secure terminals; verify polarity.
4. Capture 4–5 photos (installed node, A/B, enclosure, meter terminals).
5. Run validations in `validations/` before leaving site.

## Outputs
- Installation report (CRM comment + evidence photos)
