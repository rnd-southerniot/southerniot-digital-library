# SOP: ready_for_installation (RAK4630 + RAK5802 + MFM384)

## Goal
Package the node for field installation with all wiring accessories.

## Steps
1. Verify lab test completion and attach logs.
2. Pack RS-485 cable, ferrules, labels, spare glands, and mounting hardware.
3. Confirm install window and required site access are scheduled.

## Outputs
- CRM comment: ready-for-installation confirmation
