# SOP: site_survey (RAK4630 + RAK5802 + MFM384)

## Goal
Confirm meter access, RS-485 wiring feasibility, and installation location.

## Steps
1. Identify the target meter (Selec MFM384) and capture nameplate photo.
2. Confirm RS-485 terminal access and safe working conditions.
3. Estimate RS-485 run length and route; avoid noisy/high-voltage bundles.
4. Capture 4–5 photos with tags required by the manifest (nameplate, wiring area, route, power).
5. Run validations in `validations/` and document risks if warnings remain.

## Outputs
- CRM comment: site survey summary + photos + identified risks
