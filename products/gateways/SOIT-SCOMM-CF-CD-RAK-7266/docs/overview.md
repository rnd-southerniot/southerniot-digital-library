# RAK7266 CareDrive Gateway (Outdoor)

## Purpose
Provides LoRaWAN gateway connectivity for CareDrive projects.

## Defaults (Locked)
- Region: `AS923-1`
- Default mode: `basic_station`

## Supported Modes
- `basic_station` (default): preferred for managed deployments.
- `udp_packet_forwarder` (supported): use only when the network backend requires UDP PF.

## Field Notes
- Evidence photos are required at `site_survey`, `on_site_installation`, and `testing` (4–5 photos with tags).
- All state deliverables must be attached back into the CRM workstream.
