# SOP: hardware_preparation (RAK7266)

## Goal
Prepare gateway hardware and accessories for installation.

## Steps
1. Inventory: gateway, antenna, mounting kit, power gear (PoE injector / DC PSU), cables.
2. Label the unit with `project_id` and target location.
3. If LTE variant: install SIM, confirm APN details, and note ICCID/IMEI in CRM.
4. Pre-configure regional settings (must be `AS923-1`) and target backend (Basic Station by default).
5. Pack spares: fuses, weatherproofing tape, spare RJ45, cable glands.

## Outputs
- Prepared hardware list attached to CRM
