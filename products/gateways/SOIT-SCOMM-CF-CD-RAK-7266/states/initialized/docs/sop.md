# SOP: initialized (RAK7266)

## Goal
Create the gateway workstream in CRM, pick a variant, and lock success criteria.

## Steps
1. Create/confirm CRM workstream: `gateway`.
2. Set `product_id` and choose `variant_id` from `variants/`.
3. Record site identifiers: `site_id`, `project_id`, and install address.
4. Confirm default region is `AS923-1` and default mode is `basic_station`.
5. Add commissioning acceptance criteria to the CRM comment thread.

## Outputs
- CRM comment: initialization summary + selected variant
- Assignment: owner + expected install window
