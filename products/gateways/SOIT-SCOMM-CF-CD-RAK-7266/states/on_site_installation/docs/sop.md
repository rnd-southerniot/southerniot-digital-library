# SOP: on_site_installation (RAK7266)

## Goal
Install and secure the gateway at the approved mount location.

## Steps
1. Confirm the approved mount location from `site_survey`.
2. Mount enclosure; verify weatherproofing and cable glands.
3. Install antenna; ensure correct type and tight connections.
4. Connect grounding and verify continuity/safety.
5. Connect power/backhaul and power on.
6. Capture 4–5 photos (mounted unit, antenna, grounding, cable entry).
7. Run validations in `validations/` before leaving site.

## Outputs
- Installation report (CRM comment + evidence photos)
